# 版本说明

本区域用于查看 RpaAiClaw 软件本体和 MCP 服务的能力变化、优化内容、修复内容和升级提醒。

| 类型 | 页面 | 用途 |
| --- | --- | --- |
| 软件版本 | [软件版本更新说明](./software.md) | 记录 RpaAiClaw 版本变化、功能变化和升级提醒。 |
| MCP 服务版本 | [MCP 服务版本更新说明](./mcp-services.md) | 记录各 MCP 服务能力变化、兼容性和升级提醒。 |
| 技能库版本 | [技能库版本更新说明](./skills.md) | 记录各技能能力变化、适用场景和升级提醒。 |

## 说明规则

- 每次发布保留日期、版本、影响范围、变更摘要和升级提醒。
- 软件版本、MCP 服务版本和技能库版本分开记录，避免影响范围不清晰。
- 只记录已发布或准备发布的事实，不把临时开发想法写入版本说明。
