# 技能库

RpaAiClaw AI 技能库用于帮助自动化任务选择合适的能力方向。技能按使用场景划分，覆盖浏览器、桌面、应用工具、OCR、验证码、移动端、微信桥接和自动化扩展等场景。

## 技能矩阵

| 技能 | 功能定位 | 详情 |
| --- | --- | --- |
| 应用业务工具 | 文件、表格、数据处理、数据库、邮件和加密类任务 | [查看](./application.md) |
| 浏览器自动化 | 网页导航、点击、填表、数据提取和浏览器状态观察 | [查看](./browser.md) |
| 桌面自动化 | 窗口、控件、截图、键鼠输入、图像匹配和桌面观察 | [查看](./desktop.md) |
| 本地集成 MCP | 内置工具协调、元素引用解析和浏览器扩展桥接 | [查看](./local-mcp.md) |
| 移动端自动化 | 移动设备连接、元素查找、点击、输入和滑动 | [查看](./mobile.md) |
| OCR 文本识别 | 图片文字识别、找字、点击文本和文本区域高亮 | [查看](./ocr.md) |
| 自动化扩展运行 | Python、Node 和本地扩展任务运行 | [查看](./runtime.md) |
| 验证码识别 | 验证码区域识别、识别结果输入和表单辅助 | [查看](./verification.md) |
| 微信桥接工具 | 微信消息监听、文本发送、媒体发送和会话辅助 | [查看](./wechat.md) |

## 版本说明

每个技能都有独立版本说明页，用于记录能力变化、优化内容、修复内容和升级提醒。
