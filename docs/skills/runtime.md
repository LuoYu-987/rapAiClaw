# 自动化扩展运行

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

自动化扩展运行为 Python、Node 和本地扩展任务提供运行与依赖管理能力，适合需要自定义处理逻辑的自动化场景。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| Python 扩展 | 支持 Python 自动化扩展任务。 |
| Node 扩展 | 支持 Node 自动化扩展任务。 |
| 本地扩展 | 支持本地高性能扩展能力。 |
| 依赖管理 | 支持扩展任务所需依赖查看、安装、更新和移除。 |
| 运行观察 | 支持运行状态、输出和错误信息观察。 |

## 版本说明

记录自动化扩展运行在扩展任务、依赖管理和运行稳定性上的变化。
