# 验证码识别

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

验证码识别用于识别验证码图片、屏幕固定区域或控件相对区域中的验证码，并可辅助表单输入。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 图片验证码识别 | 支持已有验证码图片识别。 |
| 区域验证码识别 | 支持屏幕区域截图并识别验证码。 |
| 相对区域识别 | 支持基于控件位置识别验证码区域。 |
| 自动输入辅助 | 支持识别后将结果填入目标输入区域。 |
| 登录场景辅助 | 适合登录、注册和表单提交中的验证码处理。 |

## 版本说明

记录验证码识别在验证码类型、识别准确率、输入辅助和兼容性上的变化。
