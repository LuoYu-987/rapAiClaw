# 微信桥接工具

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

微信桥接工具用于微信消息监听、文本消息发送、媒体消息发送和会话辅助，适合微信场景下的消息自动化。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 状态观察 | 支持查看桥接状态、监听状态和最近回复目标。 |
| 消息监听 | 支持微信消息监听和消息来源过滤。 |
| 文本发送 | 支持向指定联系人或最近联系人发送文本消息。 |
| 媒体发送 | 支持图片、视频、文件等媒体消息发送。 |
| 会话辅助 | 支持复用最近消息上下文进行回复。 |

## 版本说明

记录微信桥接工具在消息监听、消息发送、媒体处理和会话辅助能力上的变化。
