# 桌面自动化

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

桌面自动化用于真实 Windows 桌面操作，适合窗口管理、控件识别、截图、坐标定位、键鼠输入、区域选择、录制回放和图像匹配。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 窗口管理 | 支持窗口列表、窗口状态和应用窗口操作。 |
| 控件观察 | 支持控件树、控件坐标、控件信息和高亮确认。 |
| 输入自动化 | 支持鼠标点击、移动、滚动、拖拽、文本输入和快捷键。 |
| 截图区域 | 支持桌面截图、区域截图和屏幕区域选择。 |
| 图像匹配 | 支持图像查找、多色匹配和视觉定位。 |

## 版本说明

记录桌面自动化在窗口、控件、输入、截图和图像匹配能力上的变化。
