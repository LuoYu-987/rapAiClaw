# 应用业务工具

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

应用业务工具用于结构化业务操作，适合文件处理、表格处理、数据计算、数据库查询、邮件发送、编码转换和加密摘要等任务。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 文件处理 | 支持文件读取、写入、搜索、复制、移动、压缩和下载。 |
| 数据处理 | 支持 JSON、CSV、文本、列表、日期、正则和类型转换。 |
| 表格处理 | 支持 Excel 数据读取、写入和整理。 |
| 数据库工具 | 支持数据库查询类任务。 |
| 邮件工具 | 支持邮件发送和通知场景。 |
| 加密编码 | 支持编码转换、哈希和 Base64 等处理。 |

## 版本说明

记录应用业务工具在文件、数据、表格、数据库、邮件和加密能力上的变化。
