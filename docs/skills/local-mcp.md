# 本地集成 MCP

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

本地集成 MCP 用于协调 RpaAiClaw 内置工具、浏览器扩展桥接、元素库引用解析和跨工具链组合能力。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 内置工具协调 | 协调多个本地工具能力完成组合任务。 |
| 浏览器扩展桥接 | 将浏览器扩展能力接入自动化流程。 |
| 元素引用解析 | 支持浏览器元素、桌面元素和图片元素引用转换。 |
| 跨能力组合 | 支持桌面、OCR、验证码、浏览器和变量能力协同。 |

## 版本说明

记录本地集成 MCP 在工具协调、元素引用和浏览器扩展桥接能力上的变化。
