# 浏览器自动化

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

浏览器自动化用于网页导航、页面观察、点击、输入、上传、表格提取、控制台信息查看和网络请求观察等场景。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 页面导航 | 支持打开页面、切换页面、关闭页面和处理弹窗。 |
| 页面观察 | 支持页面快照、元素识别、截图和页面状态读取。 |
| 输入交互 | 支持点击、填写、按键、拖拽、上传和表单处理。 |
| 数据提取 | 支持页面数据、表格、下载和剪贴板相关能力。 |
| 调试观察 | 支持控制台、网络请求和性能观察。 |

## 版本说明

记录浏览器自动化在页面操作、数据提取、调试观察和兼容性上的变化。
