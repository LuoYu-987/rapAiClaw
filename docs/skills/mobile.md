# 移动端自动化

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

移动端自动化用于移动设备和移动应用操作，适合设备连接、会话管理、元素查找、点击、输入、滑动、截图和上下文切换。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 设备连接 | 支持移动设备或模拟器连接状态检查。 |
| 会话管理 | 支持创建、查看和结束移动端自动化会话。 |
| 元素操作 | 支持查找、点击、输入和上下文切换。 |
| 手势操作 | 支持坐标点击、滑动和移动端交互。 |
| 观察能力 | 支持页面结构和截图观察。 |

## 版本说明

记录移动端自动化在设备连接、元素操作、手势操作和兼容性上的变化。
