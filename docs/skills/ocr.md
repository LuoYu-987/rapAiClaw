# OCR 文本识别

[查看技能库版本说明](../releases/skills.md)

## 技能介绍

OCR 文本识别用于识别图片、屏幕、区域或控件中的文字内容，并支持根据文字查找、点击、悬停和高亮。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 文本识别 | 支持图片、屏幕、区域和控件文本识别。 |
| 文本查找 | 支持按文本内容、匹配方式和顺序定位文本区域。 |
| 文字点击 | 支持识别指定文字并执行点击。 |
| 文本高亮 | 支持高亮识别结果，方便确认位置。 |
| 场景辅助 | 可辅助桌面、浏览器和验证码相关流程。 |

## 版本说明

记录 OCR 文本识别在识别准确率、识别范围、文本定位和交互能力上的变化。
