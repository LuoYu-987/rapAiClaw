# Desktop MCP 管理页

<span class="status">默认启用</span>

[查看版本更新说明](../releases/mcp/desktop.md)

## 服务介绍

Desktop MCP 是 RpaAiClaw 的桌面自动化核心服务，负责窗口状态、控件选择、截图、输入自动化、图像匹配、应用管理和系统信息采集。

## 功能范围

| 工具族 | 说明 |
| --- | --- |
| 应用管理 | 启动应用、识别窗口、维护窗口状态。 |
| 桌面状态 | 获取桌面应用、窗口层级和状态分类。 |
| 控件选择 | 选择控件、截图、录制、回放和路径解析。 |
| 输入自动化 | 点击、拖拽、滚动、热键、输入法和文本输入。 |
| 图像匹配 | 根据图片特征定位桌面目标。 |
| 截图与系统信息 | 获取桌面截图和基础系统状态。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用。 |
| 新增能力 | 记录新增的桌面操作、控件识别或图像匹配能力。 |
| 优化内容 | 记录窗口识别、截图稳定性、输入动作可靠性等优化。 |
| 兼容性 | 记录不同系统、窗口类型和权限环境下的影响。 |
| 升级提醒 | 说明用户升级后需要关注的桌面权限或使用习惯变化。 |
