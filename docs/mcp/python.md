## 服务介绍

Python MCP 提供 Python 自动化扩展、依赖管理和数据处理能力，支撑流程中的扩展任务。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 扩展任务执行 | 在流程中运行 Python 扩展任务。 |
| 包管理 | 管理扩展任务所需依赖。 |
| 自动化扩展 | 支持用户扩展流程处理能力。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用。 |
| 新增能力 | 记录新增扩展任务、依赖管理或数据处理能力。 |
| 优化内容 | 记录扩展任务稳定性、错误提示和依赖处理体验优化。 |
| 兼容性 | 记录依赖包和运行环境变化带来的影响。 |
| 升级提醒 | 说明升级后扩展任务行为或依赖管理方式变化。 |
