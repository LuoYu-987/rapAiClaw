## 服务介绍

Browser Extension MCP 提供浏览器扩展桥接能力，用于在浏览器页面内执行元素识别、页面操作、数据提取和动作录制。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 元素快照 | 获取页面结构、定位器和可交互元素。 |
| 等待与输入 | 等待元素、输入文本、点击和高亮。 |
| 元素选择 | 支持元素选择、定位器生成和调试辅助。 |
| 数据提取 | 支持页面表格和结构化数据提取。 |
| 动作录制 | 支持浏览器动作录制链路。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |

| 优化内容 | 记录页面连接稳定性、元素定位准确性和操作反馈优化。 |

| 升级提醒 | 说明用户升级扩展或启用桥接能力时需要关注的事项。 |
