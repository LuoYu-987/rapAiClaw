## 服务介绍

Mobile MCP 基于 Appium 提供移动端自动化能力，用于 App 或移动浏览器的操作、调试和流程编排。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| Appium 自动化 | 连接移动设备或模拟器并执行自动化操作。 |
| 移动端元素操作 | 支持定位、点击、输入、滑动等移动端动作。 |
| 流程集成 | 作为 MCP 工具被 BPMN 节点调用。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用。 |
| 新增能力 | 记录新增移动端连接、元素操作或设备控制能力。 |
| 优化内容 | 记录设备连接稳定性、元素定位和操作反馈优化。 |
| 兼容性 | 记录不同移动系统、设备类型和应用场景的影响。 |
| 升级提醒 | 说明升级后移动端连接方式或权限要求变化。 |
