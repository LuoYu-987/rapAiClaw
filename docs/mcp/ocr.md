## 服务介绍

OCR MCP 提供文字识别能力，服务于截图识别、验证码辅助识别、桌面和浏览器自动化中的视觉解析场景。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 图片文字识别 | 识别截图或图片中的文字内容。 |
| 识别质量评估 | 关注识别准确率、耗时和复杂图片适配情况。 |
| 自动化辅助 | 为桌面、浏览器和验证码场景提供视觉文字识别能力。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用。 |
| 新增能力 | 记录新增的识别语言、图片类型或场景支持。 |
| 优化内容 | 记录识别准确率、速度和复杂图片适配优化。 |
| 兼容性 | 记录不同图片来源、分辨率和系统环境下的影响。 |
| 升级提醒 | 说明识别效果变化和用户需要关注的使用限制。 |
