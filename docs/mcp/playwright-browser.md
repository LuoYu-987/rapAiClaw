## 服务介绍

Playwright Browser MCP 基于 Playwright 提供浏览器自动化能力，包含拖拽、表单、截图和调试类工具。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 页面控制 | 打开页面、定位元素、点击、输入、截图。 |
| 拖拽能力 | 支持浏览器页面内的拖拽类操作。 |
| 调试辅助 | 支持页面状态、截图和定位结果排查。 |
| 表单处理 | 支持常见表单输入、提交和状态检查。 |

## 版本说明

| 记录项 | 说明 |
