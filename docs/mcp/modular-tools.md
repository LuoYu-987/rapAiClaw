## 服务介绍

Modular Tools 是内置模块化工具集合，用于提供主应用内可直接调用的工具能力。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 内置工具 | 提供主应用内的常用工具能力。 |
| 工具桥接 | 与 MCP 工具列表、调用、取消和窗口控制协作。 |
| 流程集成 | 可作为 BPMN MCP 工具节点的一部分。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用。 |
| 新增能力 | 记录新增内置工具、窗口控制或流程集成能力。 |
| 优化内容 | 记录工具调用体验、取消操作和错误提示优化。 |
| 兼容性 | 记录与软件版本和流程节点的兼容关系。 |
| 升级提醒 | 说明升级后内置工具行为变化。 |
