## 服务介绍

Verification Code MCP 专注验证码识别能力，通常作为流程中的辅助工具按需连接，不作为默认自启服务。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 验证码识别 | 对验证码图片或截图片段执行识别。 |
| 流程辅助 | 支持登录、注册、表单提交等需要验证码输入的自动化节点。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用，不默认自启。 |
| 新增能力 | 记录新增验证码类型或识别场景。 |
| 优化内容 | 记录识别准确率、识别速度和失败提示优化。 |
| 兼容性 | 记录不同验证码样式和页面场景的影响。 |
| 升级提醒 | 说明升级后验证码识别策略或使用限制变化。 |
