## 服务介绍

Browser MCP 是浏览器自动化服务，提供浏览器状态、页面操作和自动化编排能力。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 浏览器连接 | 连接或管理浏览器实例。 |
| 页面操作 | 执行导航、点击、输入、截图等浏览器动作。 |
| 工具调用 | 作为 MCP 工具被流程调用。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认关闭。 |
| 新增能力 | 记录新增浏览器连接、页面操作或状态读取能力。 |
| 优化内容 | 记录页面操作稳定性、等待策略和错误提示优化。 |
| 兼容性 | 记录不同浏览器版本和页面类型下的影响。 |
| 升级提醒 | 说明用户启用服务后需要关注的浏览器环境变化。 |
