## 服务介绍

Wechat Bridge MCP 是微信桥接服务，用于对接微信相关运行时和自动化能力。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 微信桥接 | 对接微信运行环境和相关操作能力。 |
| 会话辅助 | 支持微信相关会话和状态管理。 |
| 自动化扩展 | 为微信场景提供流程自动化能力。 |
