## 服务介绍

Node MCP 提供自动化扩展、包管理和工具扩展能力，适合流程中的扩展任务。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 扩展任务执行 | 运行 Node 扩展任务。 |
| 包管理 | 管理扩展任务所需依赖。 |
| 工具扩展 | 为流程节点提供扩展能力。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认关闭。 |
| 新增能力 | 记录新增扩展任务、依赖管理或工具扩展能力。 |
| 优化内容 | 记录扩展任务稳定性、错误提示和依赖处理体验优化。 |
| 兼容性 | 记录依赖包和运行环境变化带来的影响。 |
| 升级提醒 | 说明启用或升级后扩展任务行为变化。 |
