## 服务介绍

C++ MCP 是本地高性能工具服务，用于承载需要更高性能或本地能力的工具模块。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| 本地工具 | 执行本地工具能力。 |
| 图像相关能力 | 支持本地图像处理能力扩展。 |
| 性能增强 | 适合对处理速度和本地资源访问要求较高的场景。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认关闭。 |
| 新增能力 | 记录新增本地工具、图像处理或性能增强能力。 |
| 优化内容 | 记录处理速度、资源占用和稳定性优化。 |
| 兼容性 | 记录不同系统环境下的能力差异。 |
| 升级提醒 | 说明启用或升级后对本地环境的影响。 |
