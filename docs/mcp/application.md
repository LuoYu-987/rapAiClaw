## 服务介绍

Application MCP 提供应用级通用工具，服务于流程编排中的文件处理、数据转换、Excel、数据库、邮件和软件激活场景。

## 功能范围

| 工具族 | 说明 |
| --- | --- |
| 文件处理 | 读写、复制、移动、删除、搜索、压缩和下载文件。 |
| 编码与加密 | 处理编码转换、摘要和加密相关操作。 |
| 数据计算 | 处理文本、表格、日期、列表、正则和类型转换。 |
| 表格处理 | 读取、写入和整理 Excel 数据。 |
| 数据库工具 | 连接数据库并执行查询类任务。 |
| 邮件工具 | 支持邮件发送和通知场景。 |
| 软件激活 | 支持软件授权和激活相关辅助能力。 |

## 版本说明

| 记录项 | 说明 |
| --- | --- |
| 当前状态 | 默认启用。 |
| 新增能力 | 记录新增的文件、数据、表格、数据库或邮件工具能力。 |
| 优化内容 | 记录数据处理准确性、文件操作稳定性和邮件发送体验优化。 |
| 兼容性 | 记录文件格式、数据库类型和邮件服务差异带来的影响。 |
| 升级提醒 | 说明用户升级后需要关注的数据格式或权限变化。 |
