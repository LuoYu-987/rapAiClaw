# MCP 服务功能介绍

RpaAiClaw 的 MCP 能力由多类服务组成，覆盖桌面自动化、应用工具、OCR、验证码、浏览器自动化、移动端自动化、自动化扩展、微信桥接和内置工具。每个服务都有独立介绍页，用于说明服务定位、功能范围和版本变化。

## 服务矩阵

| 服务 | 功能定位 | 状态 | 管理页 |
| --- | --- | --- | --- |
| Desktop MCP | 桌面自动化核心服务 | 默认启用 | [Desktop MCP](./desktop.md) |
| Application MCP | 应用级通用工具服务 | 默认启用 | [Application MCP](./application.md) |
| OCR MCP | 文字识别服务 | 默认启用 | [OCR MCP](./ocr.md) |
| Verification Code MCP | 验证码识别服务 | 默认启用，不默认自启 | [Verification Code MCP](./verification.md) |
| Browser MCP | 浏览器自动化服务 | 默认关闭 | [Browser MCP](./browser.md) |
| Browser Extension MCP | 浏览器扩展桥接服务 | 默认启用，不默认自启 | [Browser Extension MCP](./browser-extension.md) |
| Selenium Browser MCP | Selenium 浏览器自动化服务 | 默认关闭 | [Selenium Browser MCP](./selenium-browser.md) |
| Playwright Browser MCP | Playwright 浏览器自动化服务 | 默认关闭 | [Playwright Browser MCP](./playwright-browser.md) |
| Python MCP | Python 自动化扩展服务 | 默认启用 | [Python MCP](./python.md) |
| Node MCP | Node 自动化扩展服务 | 默认关闭 | [Node MCP](./node.md) |
| Mobile MCP | 移动端自动化服务 | 默认启用 | [Mobile MCP](./mobile.md) |
| C++ MCP | 本地高性能工具服务 | 默认关闭 | [C++ MCP](./cpp.md) |
| Wechat Bridge MCP | 微信桥接服务 | 独立维护 | [Wechat Bridge MCP](./wechat-bridge.md) |
| Modular Tools | 内置模块化工具集合 | 默认启用 | [Modular Tools](./modular-tools.md) |
| Test MCP Example | 示例服务 | 示例用途 | [Test MCP Example](./test-example.md) |

## 版本说明

每个 MCP 服务版本说明应记录服务版本、能力变化、兼容性、影响范围和升级提醒。服务版本说明只面向用户和运维视角。
