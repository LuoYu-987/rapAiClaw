## 服务介绍

Selenium Browser MCP 基于 Selenium 生态提供浏览器自动化能力，适合需要 WebDriver 兼容性的流程场景。

## 功能范围

| 能力 | 说明 |
| --- | --- |
| WebDriver 自动化 | 通过 Selenium 执行浏览器动作。 |
| 页面交互 | 支持导航、定位、点击、输入和状态读取。 |
| 兼容场景 | 适合需要 WebDriver 兼容性的浏览器流程。 |
