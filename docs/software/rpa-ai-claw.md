## 软件介绍

RpaAiClaw 是桌面自动化工作台，面向流程自动化、浏览器自动化、桌面自动化、移动端自动化和 AI 工具编排场景。软件将流程设计、工具调用、运行监控、任务调度、授权管理和版本更新整合到统一工作台中。

## 核心能力

| 能力域 | 说明 |
| --- | --- |
| 工作流与流程库 | 管理 BPMN 流程、分类、导入导出、运行记录和调试入口。 |
| 导航式设计器 | 提供 BPMN 画布、节点配置、运行态底栏、右侧 MCP 工具面板。 |
| 浏览器自动化 | 通过浏览器扩展桥、Playwright/Selenium/Browser MCP 和动作录制窗口完成页面操作。 |
| 桌面自动化 | 通过 Desktop MCP 管理窗口、控件、截图、输入、图像匹配和桌面状态。 |
| AI 工作区 | 管理 AI 配置、提示调用、对话记录和 AI 引用窗口。 |
| 多窗口运行态 | 支持 Quick Pane、人工交互、运行态、AI 引用、浏览器录制、桌面录制等窗口。 |
| 系统工具台 | 管理 MCP、日志、授权、偏好设置、更新和浏览器扩展。 |
