## 功能范围

- 流程设计：通过流程画布组织自动化步骤。
- 自动化执行：支持浏览器、桌面、移动端和本地工具能力。
- MCP 服务：通过 MCP 服务扩展文件、表格、OCR、验证码、脚本扩展等工具能力。
- 运行管理：提供运行状态、日志、任务调度和人工交互能力。
- 版本说明：记录软件版本变化、能力优化和升级提醒。
